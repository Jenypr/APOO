# UC01 – Gerenciar Vídeos

## Ator
Criador de Conteúdo

## Descrição
Permite ao criador cadastrar, editar ou remover vídeos do sistema.

## Pré-condições
Usuário autenticado no sistema.

## Pós-condições
Vídeo cadastrado, atualizado ou removido.

## Fluxo Principal
1. Usuário acessa a lista de vídeos.
2. Seleciona a opção de gerenciamento.
3. Informa os dados do vídeo.
4. O sistema valida as informações.
5. O sistema salva os dados.

## Fluxos Alternativos
3.1 Dados inválidos → sistema solicita correção.

## Exceções
Falha de conexão com o banco de dados.

Diagrama de sequência e comunicação.
