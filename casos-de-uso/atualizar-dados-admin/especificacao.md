# Caso de Uso: Atualizar Dados do Administrador

## Atores Envolvidos
- Administrador

---

## Pré-condições
- O administrador deve estar autenticado no sistema.

---

## Pós-condições
- Os dados do administrador são atualizados no sistema.

---

## Fluxo Principal – Atualizar Dados
1. O administrador acessa a tela de atualização de dados.  
2. O sistema exibe os dados atuais do administrador.  
3. O administrador altera os dados desejados.  
4. O sistema valida os dados informados.  
5. O sistema atualiza os dados do administrador no banco de dados.  
6. O sistema exibe uma mensagem de confirmação.

---

## Fluxo Alternativo – Dados inválidos
1. O administrador informa dados inválidos.  
2. O sistema exibe uma mensagem de erro.  
3. O sistema solicita a correção das informações.
