# UC02 – Gerenciar Comentários

## Ator
Criador de Conteúdo

## Descrição
Permite visualizar, responder ou remover comentários dos vídeos.

## Pré-condições
Usuário autenticado.

## Pós-condições
Comentário tratado.

## Fluxo Principal
1. Usuário acessa comentários.
2. Seleciona comentário.
3. Responde ou remove.
4. Sistema registra a ação.

## Fluxos Alternativos
3.1 Comentário ofensivo → remoção imediata.

## Exceções
Falha na comunicação com a API do YouTube.

Diagrama de sequência e comunicação.
