# UC03 – Gerar Relatórios

## Ator
Administrador

## Descrição
Permite gerar relatórios de desempenho do canal.

## Pré-condições
Administrador autenticado.

## Pós-condições
Relatório gerado.

## Fluxo Principal
1. Usuário seleciona período.
2. Sistema coleta dados.
3. Sistema processa métricas.
4. Sistema exibe relatório.

## Fluxos Alternativos
2.1 Período inválido → sistema solicita correção.

## Exceções
Falha no processamento dos dados.

## Diagrama Associado
Diagrama de sequência e comunicação.
