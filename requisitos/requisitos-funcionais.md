# Requisitos Funcionais

RF01 – O sistema deve permitir o cadastro de usuários.  
RF02 – O sistema deve permitir autenticação por login e senha.  
RF03 – O sistema deve permitir o cadastro de canais do YouTube.  
RF04 – O sistema deve permitir o gerenciamento de vídeos.  
RF05 – O sistema deve permitir o gerenciamento de comentários.  
RF06 – O sistema deve permitir a geração de relatórios.
