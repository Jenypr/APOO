# Requisitos Não Funcionais

RNF01 – O sistema deve garantir segurança das informações.  
RNF02 – O sistema deve possuir disponibilidade contínua.  
RNF03 – O sistema deve possuir interface intuitiva.  
RNF04 – O sistema deve apresentar desempenho adequado.  
RNF05 – O sistema deve ser compatível com navegadores modernos.
